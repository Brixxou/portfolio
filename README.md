# Portfolio de brixxou

Depot du site personnel, oriente DevOps et build in public.

- Presentation du parcours et des pratiques DevOps.
- Notes d'apprentissage et projets en cours.
- Acces rapide aux profils sociaux, au CV et a la stack.
- Notes publiques chargees automatiquement depuis `data/notes.json`.

## Ajouter une note

1. Ouvrir `data/notes.json`.
2. Ajouter un objet avec les cles `tag`, `title`, `excerpt`, `url`.
3. Sauvegarder : la page lira ce fichier et generera les cartes automatiquement.

Le site est 100 % statique : ouvrir `index.html` en local ou le deployer sur une plateforme de pages statiques (GitHub Pages, Netlify, Vercel, S3/CloudFront).
